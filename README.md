# Student Management System

## 📌 Project Overview

The **Student Management System** is a simple Frontend and Backend based application that uses flask for backend and html ,css , javascript for backend. 
It is designed to manage student records efficiently by performing basic database operations such as **adding**, **viewing**, **searching**, **updating**, and **deleting** student information.

This project demonstrates the practical implementation of:

- **Python programming**
- **Flask and pydocs**
- **SQL Server database integration**
- **CRUD operations**
- **UI application that control UI with DATA**

The project is usefull for understand about the joins for database and flask to understand about the backend for real world application.

---

## 🎯 Objective of the Project

The main objective of this project is to build a simple database management system using Python.  
It helps in understanding how to:

- Connect Python with a database
- Perform database operations using SQL queries
- Use pydocs for communication between python and SQL Server
- Create a menu-driven application for record management
- Implement CRUD functionality in a practical project

This Project is Considered as a simple DBMS Project for a college submission.

---

## ✨ Features

The system provides the following functionalities:

### ✅ Add Student
Allows the user to insert a new student record into the database.

### ✅ View Students
Displays all student records stored in the database.

### ✅ Search Student
Searches for a student using their unique ID.

### ✅ Update Student
Updates the details of an existing student.

### ✅ Delete Student
Deletes a student record from the database.

### ✅ Duplicate ID Check
Prevents insertion of student records with duplicate IDs.

### ✅ Database Connectivity
Uses pydocs to connect the Java application with SQL Server.
